GitHub Setup
ℹ️ Connect GitHub so ALL visitors (not just you) see your products and video. Free and takes 5 minutes.

🐙 GitHub Connection

Token stored in your browser only. Never shared.

📋 Setup Steps

  1. Go to github.com → New repository → name: farnaz → Public
  2. Inside repo → Add file → Create new file → name: data.json → content: {"products":[],"settings":{}} → Commit
  3. GitHub → Settings → Developer Settings → Personal access tokens → Fine-grained → Generate new token
  4. Give token Contents: Read & Write on your repo → Copy token
  5. Fill in the fields above → click Save & Connect
  6. Click Push All Data Now to make your products visible to everyone!
